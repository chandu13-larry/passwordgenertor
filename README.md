🔐 PyPassword Generator

A simple Python script that generates a random password based on user preferences for letters, numbers, and symbols.

📌 Features
Customizable number of:
Letters (uppercase + lowercase)
Numbers
Symbols
Randomized password generation
Password strength feedback
Input validation for numeric values

🛠️ How It Works
The user is prompted to enter:
Number of letters
Number of symbols
Number of numbers
The program:
Randomly selects characters from predefined lists
Combines them into a password
Shuffles the password for randomness
Displays:
Generated password
Strength assessment (Weak / Medium / Strong)

🚀 Getting Started
Prerequisites
Python 3.x installed
Run the Script
python password_generator.py

🧾 Example Usage
